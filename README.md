# Diamond Markets - Trading Platform

## Setup Instructions

### Required Images

Please add the following images to the `public` folder:

1. **Vfx-logo_white.png** - Your white logo (height: ~25px)
2. **newOrder.png** - New order button icon (width: ~18px)

The path should be:
- `public/Vfx-logo_white.png`
- `public/static/media/newOrder.75726de54c82905fd8ac.png`

### Running the Project

```bash
npm install
npm run dev
```

### Color Palette Used

- **Background Dark**: `#0f0f1e` - Main background
- **Navbar Background**: `#1a1a2e` - Navbar background
- **Primary Blue**: `#3b82f6` to `#2563eb` - Buttons and accents
- **Text Colors**: 
  - White: `#ffffff`
  - Gray 300: `#d1d5db`
  - Gray 400: `#9ca3af`
  - Gray 700: `#374151`
  - Gray 800: `#1f2937`

### Features

- Responsive navbar with Tailwind CSS
- Modern dark theme matching the original design
- Smooth hover transitions
- Mobile-friendly layout
